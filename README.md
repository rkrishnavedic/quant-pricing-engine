# Quant Pricing Engine

A modular Python library for pricing financial derivatives using:

- Black-Scholes analytical solutions
- Monte Carlo simulation
- Numerical discretization methods

## Goal

To build a production-style pricing engine demonstrating:
- Analytical pricing
- Simulation-based pricing
- Model validation & convergence analysis

## Python Project setup
### 1. Environment Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Run Notebook Environment
```bash
pip install jupyterlab
jupyter lab
```
